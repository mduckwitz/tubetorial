# Tube-torial Codebase

This repository contains the source code and examples for the **Tube-torial**, a guide to mastering the Transport for London (TfL) Unified API.

## Contents

*   **`london_tube_tutorial.md`**: The comprehensive written guide.
*   **`tube_status.py`**: Basic script to check the status of all Tube lines.
*   **`station_search.py`**: Utility to find the NaPTAN ID for a specific station name.
*   **`live_arrivals.py`**: Real-time arrival board for a specific station.
*   **`check_crowds.py`**: Advanced script to check crowding profiles.

## Setup

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the scripts (e.g., `python tube_status.py`).
